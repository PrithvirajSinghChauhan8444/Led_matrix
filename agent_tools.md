# Antigravity Agentic Toolbox: Final Daily & Fun Set

This is the finalized list of tools that can be implemented for the Antigravity assistant to assist daily and keep things interesting.

## 1. Digital Wellbeing & Real-World Link
*   **`wellness_coach`**: 
    *   **What it does**: Tracks active PC time and pings the user (via desktop or phone) to stretch or hydrate.
*   **`relay_ping`**: 
    *   **What it does**: The bridge between PC and phone. Sends alerts (Wellness, Weather, BrainDumps) directly to the user via Telegram/Pushbullet.
*   **`weather_advisor`**: 
    *   **What it does**: Monitors local conditions. Provides a morning brief and sends a "Rain Check" ping if precipitation is expected.

## 2. Command & Launch (System Actions)
*   **`launch_app`**: 
    *   **What it does**: Instantly launches any application on the system (Spotify, Discord, Steam, etc.) on command.
*   **`session_setup`**: 
    *   **What it does**: A "one-command" setup for specific moods. "Setup for Movie Night" or "Setup for Gaming."

## 3. Fun & Games (Creative Actions)
*   **`rpg_portal`**: 
    *   **What it does**: Starts an interactive, text-based RPG or adventure game where choices determine the story.
*   **`meme_lab`**: 
    *   **What it does**: Creates custom memes. Uses web search for current trends and built-in image generation for visuals.
*   **`riddle_master`**: 
    *   **What it does**: Challenges the user with a daily riddle and tracks streaks.
*   **`dream_canvas`**: 
    *   **What it does**: Turns text descriptions of dreams/ideas into high-quality digital art using built-in image generation tools.
*   **`fortune_oracle`**: 
    *   **What it does**: A futuristic "Magic 8-Ball" for playful daily predictions.

## 4. Quick Utilities
*   **`brain_dump`**: 
    *   **What it does**: Instantly appends a quick note or idea to a `brain_dump.txt` file.
*   **`message_polisher`**: 
    *   **What it does**: Refines the tone of rough draft messages and copies them back to the clipboard.

---
> [!NOTE]
> These tools are designed to be implemented as simple Python or Bash scripts that interface with the agent's command-line capabilities.
